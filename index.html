Simple Selectors

Selector Description Example
Type
element		 Selects an element based on its HTML tag p { font-size: 18px; }
Class
.class		 Selects every element with this class attribute .link { font-weight: bold; }
Multiple classes
.class1.class2.class3 		Selects every element with these classes within its class attribute .menu.option.disabled { opacity: 0.5; }
Specific elements by class
element.class 		Selects elements of the specified type with this class attribute p.important { color: red; }
ID
#id		 Selects an element based on its ID #logo { margin: 10px; }
Universal
*		 Selects all elements * { font-family: Helvetica; }

Grouping selector

Selector Description Example
Grouping selector
element, element		 Selects multiple elements at once h2, p, .search { font-style: italics; }

Combinators

Selector Description Example
Descendant
element element		 Selects all descendants of the specified element main p { line-height: 1.6; }
Child
element > element		 Selects only the direct descendants of the specified element ul > li { text-decoration: underline; }
Adjacent/next sibling
element + element 		Selects any element which is directly after the specified element h1 + p { font-weight: bold; }
General/subsequent sibling
element ~ element 		Selects any element which comes after the specified element and shares the same parent p ~ img { display block; }
