Introduction

Selectors are used to identify and target one or more HTML elements on a page. These elements can then be styled by using CSS rules. There are many different types of selectors we can use.

Basic Selectors

Selector Description Code
Type Selects an element based on its HTML tag h1
Class Selects elements based on their class .class
ID Selects an element based on its ID #id
Universal (*) Selects all elements *
