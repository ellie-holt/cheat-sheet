Simple Selectors

Selector Description Example
Type element Selects an element based on its HTML tag h1
Class .class Selects every element with this class attribute .class
Multiple classes .class1.class2.class3 Selects every element with these classes within its class attribute .class1.class2.class3
Specific elements by class element.class Selects elements of the specified type with this class attribute h1.class
ID #id Selects an element based on its ID #id
Universal * Selects all elements *

Combinators

Selector Description Example
Group/selector list element, element (whitespace after comma optional) Selects multiple elements at once h1, p, div
Descendant element element Selects all descendants of the specified element h1 p
Child element > element Selects only the direct descendants of the specified element h1 > p
Adjacent/next sibling element + element Selects any element which is directly after the specified element h1 + p
General/subsequent sibling element ~ element Selects any element which comes after the specified element and shares the same parent h1 ~ p
