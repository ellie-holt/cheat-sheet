Simple Selectors

Selector Description Example
Type
element		 Selects an element based on its HTML tag p { /* Styles applied to all <p> elements */ font-size: 18px; }
Class
.class		 Selects every element with this class attribute .link { /* Styles applied to elements with class 'link' */ font-weight: bold; }
Multiple classes
.class1.class2.class3 		Selects every element with these classes within its class attribute .menu.option.disabled { /* Styles applied to elements with all three classes */ opacity: 0.5; }
Specific elements by class
element.class 		Selects elements of the specified type with this class attribute p.important { /* Styles applied to <p> elements with class 'important' */ color: red; }
ID
#id		 Selects an element based on its ID #logo { /* Styles applied to elements with ID 'logo' */ margin: 10px; }
Universal
*		 Selects all elements * { /* Styles applied to all elements */ font-family: Helvetica; }

Grouping selector

Selector Description Example
Grouping selector
element, element		 Selects multiple elements at once h2, p, .search { /* Styles applied to <h2> and <p> elements, and all elements with class 'search' */ font-style: italics; }

Combinators

Selector Description Example
Descendant
element element		 Selects all descendants of the specified element main p { /* Styles applied to all <p> elements inside <main> */ line-height: 1.6; }
Child
element > element		 Selects only the direct descendants of the specified element ul > li { /* Styles applied to <li> elements that are direct children of <ul> */ text-decoration: underline; }
Adjacent sibling
element + element		 Selects any element which is directly after the specified element h1 + p { /* Styles applied to <p> directly following <h1> */ font-weight: bold; }
General sibling
element ~ element		 Selects any element which comes after the specified element and shares the same parent p ~ img { /* Styles applied to <img> that follow <p> */ display: block; }

Attribute selectors

Selector Description Example
[att] Selects all elements with the specified attribute, regardless of its value img[alt] { /* Styles applied to <img> elements with an 'alt' attribute */ border: 2px solid; }
[att=val] Selects all elements with the specified attribute value input[type="submit"] { * Styles applied to <input> elements with type 'submit' */ padding: 8px 20px; }
[att~=val] Selects an element based on whether its attribute contains a specific value among multiple space-separated values [class~="featured"] { /* Styles applied to elements containing 'featured' within its class attributes */ box-shadow: 0 4px 8px grey; }
[att|=val] Selects all elements where the attribute value starts with a specified value followed by a hyphen [lang|="en"] { /* Styles applied to elements with language 'en' or 'en-' */ color: blue; }
[att^=val] Selects all elements where the attribute value starts with a specified string a[href^="https"] { /* Styles applied to <a> elements with links starting with 'https' */ text-decoration: none; }
[att$=val] Selects all elements where the attribute value ends with a specified string img[src$=".png"] { /* Styles applied to <img> elements with image sources ending in '.png' */ width: 100px; }
[att*=val] Selects all elements where the attribute value contains a specified substring [class*="btn"] { /* Styles applied to elements with class containing 'btn' */ cursor: pointer; }

Pseudo-classes

Selector Description Example
:link Selects unvisited links a:link { /* Styles applied to unvisited links */ color: blue; }
:visited Selects visited links a:visited { /* Styles applied to visited links */ color: purple; }
:hover Selects an element when the mouse is over it a:hover { /* Styles applied to links on hover */ color: blue; }
:active Selects the active link a:active { /* Styles applied to the active link */ color: red; }
:focus Selects the element that has focus input:focus { /* Styles applied to the focused input */ border-color: green; }
:checked Selects input elements that are checked, like checkboxes or radio buttons input:checked { /* Styles applied to checked input elements */ border-color: blue; }
:disabled Selects elements that are disabled button:disabled { /* Styles applied to disabled buttons */ opacity: 0.5; }
:first-child Selects the first child of its parent p:first-child { /* Styles applied to first paragraph of its parent */ font-weight: bold; }
:last-child Selects the last child of its parent div:last-child { /* Styles applied to last div of its parent */ border: 1px solid black; }
:nth-child(n) Selects elements based on their position in a group of siblings li:nth-child(odd) { /* Styles applied to odd list items */ background-color: lightgrey; }
:first-of-type Selects the first element of its type among its siblings p:first-of-type { /* Styles applied to the first paragraph among its siblings */ color: red; }
:last-of-type Selects the last element of its type among its siblings p:last-of-type { /* Styles applied to the last paragraph among its siblings */ text-decoration: underline; }
:nth-of-type() Selects elements based on their position among siblings of the same type p:nth-of-type(odd) { /* Styles applied to odd-numbered paragraphs */ font-weight: bold; }
:not(selector) Selects elements that do not match a specific selector p:not(.special) { /* Styles applied to paragraphs that are not of class 'special' */ color: grey; }

Pseudo-elements

Selector Description Example
::before Inserts content before the selected element p::before { /* Styles applied to content before paragraphs */ content: "🔹"; }
::after Inserts content after the selected element div::after { /* Styles applied to content after divs */ content: "⭐️"; }
::first-line Selects the first line of text within an element p::first-line { /* Styles applied to the first line of paragraphs */ font-style: italic; }
::last-line Selects the last line of text within an element p::last-line { /* Styles applied to the last line of paragraphs */ font-weight: bold; }
::first-letter Selects the first letter of a specified element p::first-letter { /* Styles applied to the first letter of paragraphs */ font-size: larger; }
::last-letter Selects the last letter of a specified element p::last-letter { /* Styles applied to the last letter of paragraphs */ color: green; }
::selection Styles the portion of an element that is selected by the user ::selection { /* Styles applied to selected text */ background-color: yellow; }
